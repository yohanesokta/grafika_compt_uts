# Maze Game - Code::Blocks Edition 😬

> Kalau kamu buka proyek ini di Code::Blocks, bersiaplah nostalgia ke era dinosaurus compiler 🦖

---

## Tentang Proyek
Ini adalah implementasi **Recursive Division Maze** dengan bahasa C/C++.
- Setiap sel maze punya koordinat pixel `(x1, x2, y1, y2)` → bisa langsung divisualisasi.
- Dirancang untuk **Code::Blocks** tapi jangan harap IDE modern menahanmu wkwk.

---

## Cara Compile / Run
1. Buka `projects.cbp` di **Code::Blocks**.  
2. Build project.  
3. Jalankan executable dari `bin/`.  

> Tips: Kalau mau pengalaman smooth, coba buka di **Visual Studio** atau **VSCode + MinGW**, bukan Code::Blocks 😏  

---

## Catatan Lucu
- `.cbp` files = nostalgia 2000-an  
- `obj/` = tempat semua object files bersembunyi  
- Jangan heran kalau Code::Blocks terasa lambat, itu fiturnya wkwk  
- Kalau kamu developer modern, upgrade IDE-mu sekarang juga  

---

Proyek ini bebas digunakan untuk belajar dan eksperimen. Tidak ada garansi “tidak crash di Code::Blocks”.
